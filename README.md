# سَمْت | SAMT
## Strategic Aviation Management & Transformation
### International Aviation Leadership Fellowship

A premium, bilingual, single-page website developed for **SAMT**, an international leadership fellowship designed to prepare future leaders across the aviation enterprise.

## Website Features

- Arabic-first interface with full RTL support
- Complete English language option
- Responsive desktop, tablet, and mobile layouts
- Cinematic aviation background and layered parallax motion
- Interactive leadership flight-path milestones
- Animated programme progress and counters
- Interactive national and international preparation stations
- Executive summary, purpose, selection model, competencies, governance, outputs, and approval pathway
- Keyboard navigation and reduced-motion accessibility support
- No external framework or build process required

## Main File

Rename the provided standalone file:

```text
SAMT_parallax_index.html
```

to:

```text
index.html
```

Then place it in the root of the website repository.

## GitHub Pages Deployment

1. Create a new GitHub repository.
2. Upload `index.html` to the repository root.
3. Open **Settings** in the repository.
4. Select **Pages**.
5. Under **Build and deployment**, choose **Deploy from a branch**.
6. Select the `main` branch and the `/root` folder.
7. Save and wait for GitHub to publish the website.

The published address will normally follow this format:

```text
https://USERNAME.github.io/REPOSITORY-NAME/
```
