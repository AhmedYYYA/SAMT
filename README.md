# SAMT Hybrid Website — Standalone Deployment v4.1

This package is designed specifically for simple GitHub Pages deployment from a phone or computer.

## Why this version is safer

Each HTML page contains its own:
- CSS styling
- JavaScript interactions
- desktop hero image
- mobile hero image

There is no required `assets` folder. The visual design will therefore not disappear when only the HTML pages are uploaded.

## Upload these files to the repository root

- index.html
- programme.html
- journey.html
- stations.html
- selection.html
- governance-impact.html
- SAMT_programme_brief_v1.0.pdf
- README.md

Replace the existing files with the versions in this package.

## GitHub Pages

Use:
- Branch: main
- Folder: /root

After committing, wait a few minutes and refresh:
https://ahmedyyya.github.io/SAMT/

If GitHub Pages displays an older cached version, open the page in a private browser tab or refresh after several minutes.
